# InsTag
這是一個 `Python` 爬蟲。
利用輸入 **Hashtag** 的 **關鍵字**，將會開始下載符合公開貼文的圖片。(預設資料夾於 `/save/<Hashtag>/`)

# Step
1. 安裝所需函式庫
```bash
$ pip install -r requirements
```

2. Git Clone
```bash
$ mkdir InsTag
$ cd InsTag
$ git clone https://github.com/alpaca0x0/InsTag.git
```

3. Run!
```bash
$ python3 main.py
```

# 操作
打上關鍵字即可開始下載圖片。
